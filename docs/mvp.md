# Sprint 3 MVP Definition

## MVP goal

Build the minimum working foundation needed to analyze municipal grievance data in Python and Jupyter.

## MVP scope

- create a reproducible Python project using `uv` and `pyproject.toml`
- set up a clean folder structure for data science work
- load a sample grievance dataset into Pandas
- inspect data quality and basic schema details
- perform essential cleaning steps
- produce basic summary statistics and visualizations
- document insights, assumptions, and limitations

## Out of scope for this sprint

- advanced NLP pipelines
- hotspot clustering models
- predictive modeling
- dashboard deployment

## Success criteria

The sprint is successful when a new contributor can set up the project, run the starter analysis, inspect the dataset, and understand the first round of findings from the repository alone.
